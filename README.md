# Приложение для визуализации данных

Python desktop-приложение для анализа и визуализации данных из CSV-файлов с графическим интерфейсом на PyQt6.

## Оглавление

- [Функциональность](#функциональность)
- [Пошаговое руководство](#пошаговое-руководство)

## Функциональность

### Основной функционал
- Загрузка CSV файлов - поддержка файлов любого размера
- Просмотр данных - табличное отображение первых 100 строк
- Статистический анализ - автоматическая генерация описательной статистики

### Визуализация данных
- Графики корреляции - парные диаграммы рассеяния для числовых параметров
- Тепловые карты - визуализация корреляционной матрицы
- Линейные графики - построение графиков по выбранным числовым столбцам

### Дополнительные функции
- Лог действий - полная история операций с временными метками
- Интуитивный интерфейс - вкладки для удобной навигации
- Обработка ошибок - информативные сообщения об ошибках

## Пошаговое руководство

1. **Загрузка данных**
   - Нажмите кнопку "Загрузить CSV файл"
   - Выберите файл с расширением .csv
   - Данные автоматически загрузятся и отобразятся

2. **Просмотр статистики (Вкладка 1)**
   - Просмотрите таблицу с данными
   - Изучите автоматически сгенерированную статистику
   - Обновите статистику при необходимости

3. **Анализ корреляций (Вкладка 2)**
   - Нажмите "Построить график корреляции"
   - График откроется в отдельном окне
   - Анализируйте взаимосвязи между переменными

4. **Тепловая карта (Вкладка 3)**
   - Нажмите "Построить тепловую карту корреляции"
   - Визуализируйте силу корреляций между параметрами

5. **Линейные графики (Вкладка 4)**
   - Выберите числовой столбец из выпадающего списка
   - Нажмите "Построить линейный график"
   - Анализируйте тенденции и паттерны данных

6. **Мониторинг действий (Вкладка 5)**
   - Просматривайте историю всех операций
   - Очищайте лог при необходимости
